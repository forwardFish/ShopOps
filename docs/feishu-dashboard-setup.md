## 飞书仪表盘 AI 提示词

```text
请在当前飞书多维表格 Base 里创建一个名为「经营驾驶舱_图表版」的仪表盘。不要修改任何数据表、字段、公式、记录或视图，只基于已有字段配置图表。

数据源：
1. 「经营仪表盘KPI快照表」：用于顶部 KPI 指标卡。指标字段均为公式字段，筛选期间和平台即可。
2. 「公式动态经营汇总表」：用于按日期、平台查看经营趋势、当日平台对比和明细趋势。

顶部放 KPI 指标卡，数据源选择「经营仪表盘KPI快照表」：
1. 今日销售额、今日有效销售额、今日退款金额、今日投流消耗、今日已知费用后利润：筛选期间=今日，平台=全平台总计。
2. 最近7天订单数、销售额、有效销售额、退款金额、达人佣金、投流消耗、已知费用后利润：筛选期间=最近7天，平台=全平台总计。
3. 最近30天订单数、销售额、有效销售额、退款金额、达人佣金、投流消耗、已知费用后利润：筛选期间=最近30天，平台=全平台总计。
4. ROI / 平台ROI 类卡片也使用「经营仪表盘KPI快照表」，不要在仪表盘里对每日 ROI 求和或平均。

第二行放趋势图，数据源使用「公式动态经营汇总表」，X 轴为日期字段，按日期升序：
1. 每日销售额趋势折线图，Y 轴为销售额/有效销售额，筛选平台=全平台总计。
2. 每日投流消耗趋势折线图，Y 轴为投流消耗，筛选平台=全平台总计。
3. 每日 ROI 趋势折线图，Y 轴为 ROI/平台ROI，筛选平台=全平台总计。
4. 每日利润趋势折线图，Y 轴为利润/投流后毛利/经营利润估算，筛选平台=全平台总计。

第三行放平台对比图：
1. 各平台 ROI 柱状图：维度=平台，指标=ROI/平台ROI，排除平台=全平台总计，按 ROI 从高到低排序。
2. 各平台利润条形图：维度=平台，指标=利润/投流后毛利/经营利润估算，排除平台=全平台总计，按利润从高到低排序。
3. 各平台投流消耗柱状图：维度=平台，指标=投流消耗，排除平台=全平台总计，按投流消耗从高到低排序。

第四行放数据质量和经营辅助图：
1. 数据状态占比饼图：维度=数据状态，指标=记录数。
2. 缺失项排行榜：维度=缺失项，指标=记录数。

添加 2 个切片器并尽量联动所有相关图表：
1. 日期切片器：字段选择日期/统计日期。
2. 平台切片器：字段选择平台。

整体风格要像老板经营驾驶舱，突出销售额、投流消耗、利润、ROI；布局紧凑，优先让移动端也能看清关键指标。
```
